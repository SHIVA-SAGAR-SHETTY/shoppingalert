Admin settings

Adding or removing products commits to your GitHub repo, so it needs a fine-grained personal access token scoped to this one repo (Contents: Read and write). It is stored only in this browser and never uploaded.

  1. Open GitHub → Fine-grained tokens.
  2. Repository access → Only select repositories → shoppingalert.
  3. Permissions → Repository → Contents → Read and write.
  4. Generate, copy, and paste it below.
